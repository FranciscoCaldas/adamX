# adamX
Optimizer benchmarks for the paper "AdamX: Cosine similarity meets gradient descent".

## Run
Use `scripts/run.py` and pass dataset/model/optimizer flags.

```bash
python scripts/run.py --dataset cifar10 --model cifarnet --optimizer adamx --epochs 100
```

## 

AdamX code is available in optimizers/adamx_final_version_.py
